## 🎮 使用流程

### 创建游戏文章
1. **访问首页** → 点击"创建新文章"
2. **选择模板** → 点击"🐍 插入贪吃蛇游戏模板"
3. **编辑内容** → 自定义文章标题和描述
4. **发布文章** → 完成创建

### 手动插入游戏
1. **编写文章** → 在内容中任意位置
2. **插入标记** → 输入 `[SNAKE_GAME]`
3. **发布文章** → 标记自动转换为游戏

### 游戏体验
1. **访问文章** → 包含游戏标记的文章
2. **开始游戏** → 点击"开始游戏"按钮
3. **控制移动** → 使用方向键操作
4. **获得分数** → 吃掉食物增长蛇身

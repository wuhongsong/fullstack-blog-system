## 🚀 Render部署步骤（超简单）

1. 访问：https://render.com
2. 用GitHub账号登录
3. 点击"New Web Service"
4. 选择您的GitHub仓库
5. 填写配置：
   - Build Command: `cd server && npm install`
   - Start Command: `node server/server.js`
6. 点击"Deploy"

**3分钟内完成部署！**
